  • Usuario 2 Mediante el comando git remote add "usuario3/release" "url del repositorio que se quiere mandar los cambios" y luego se hace un git push usuario3/release "nombre de la rama"
  • Usuario 3 en la rama develop se hace un git pull origin feature/contenidoHTML, se hace un merge si hace falta y cuando ya estan solucionado los errores se hace un git push origin develop
